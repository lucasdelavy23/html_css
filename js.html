JS

JavaScript é a linguagem responsável por dar comportamento e interatividade as paginas web, permitindo responder a ações do usuario (como cliques e digitação), manipular o conteudo do HTML em tempo real e se comunicar com servidores.

Em resumo, enquanto HTML estrutura e CSS estiliza, o javaScript faz a página "funcionar".
